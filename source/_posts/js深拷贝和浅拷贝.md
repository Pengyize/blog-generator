---
title: JS深拷贝和浅拷贝
date: 2018-03-21 14:12:09
tags:
---
```
var a = 1;        //        a === 1
b = a;            //        b === 1
b = 2;             //        b === 2,a === 1
```
这就是深拷贝，意思是拷贝之后，改变其中一个值的同时并不改变另一个值。

除了object以外的数据类型，赋值都是深拷贝

```
var a = {
    name: ' pyz'
}
b=a;
b.name = 'jpj'        //        这时a和b都改成了jpj
```
这就是浅拷贝，赋值时只能把地址赋过去，并没把heap内存里的东西赋给b。
